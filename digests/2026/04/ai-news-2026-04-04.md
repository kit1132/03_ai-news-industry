## 新モデル・新機能

### Google、オープンモデル「Gemma 4」を公開（4/2）
Gemini 3ベースのオープンモデルファミリー。2B/4B/26B/31Bの4サイズ、140言語超対応、最大256Kコンテキスト。音声・映像のマルチモーダル処理に対応。Apache 2.0ライセンスでHugging Face/Kaggle/Ollamaから入手可能。エッジデバイス（スマホ、Raspberry Pi等）でオフライン動作。
- https://blog.google/innovation-and-ai/technology/developers-tools/gemma-4/
- https://dataconomy.com/2026/04/03/google-releases-open-source-ai-model-gemma-4-for-developers/

### Google、Geminiを Android Autoに展開開始（4/3）
2025年11月発表のGemini on Android Autoが一般公開開始。Google Maps連携、メッセージ編集、音楽操作を会話型音声で操作可能。
- https://letsdatascience.com/news/google-expands-gemini-integration-on-android-auto-09a252f2

### Netflix、初の公開モデル「VOID」をHugging Faceで公開
Video Object and Interaction Deletion — 動画からオブジェクトを除去し、物理的相互作用（影・反射だけでなく落下等）まで再生成するモデル。オープンウェイト。
- https://huggingface.co/netflix/void-model

## プラットフォーム・ツール更新

### Microsoft 365 Copilot Chat、4/15から無料ユーザーのOfficeアプリ内利用を制限
Word・Excel・PowerPoint・OneNoteでのCopilot Chatが有料ライセンス専用に。無料ユーザーはOutlook（受信トレイ・カレンダー連携）のみ利用可。新バンドル「Microsoft 365 E7（Frontier Suite）」は5/1提供開始。
- https://blogs.imperial.ac.uk/office365/2026/03/20/copilot-chat-changes-april-2026/
- https://learn.microsoft.com/en-us/partner-center/announcements/2026-april

### Claude API — Message Batches APIのmax_tokensが300Kに拡大
Claude Opus 4.6 / Sonnet 4.6でMessage Batches APIの最大トークン数を300Kに引き上げ。長文生成・大規模コードタスク向け。また、Claude Sonnet 4.5/4の1Mコンテキストβは4/30で終了予定。
- https://releasebot.io/updates/anthropic

### Claude Code — alt-screen描画改善、名前付きサブエージェント等
フリッカーフリーのalt-screen描画、@メンション内での名前付きサブエージェント、パーミッション拡張、PowerShellサポート改善。長時間セッション安定性・Windows/音声の問題修正。
- https://releasebot.io/updates/anthropic/claude-code
