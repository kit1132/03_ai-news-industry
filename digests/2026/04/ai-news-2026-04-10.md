# AI News Digest: 2026-04-10

## 主要プレイヤーの動向

### Meta、独自モデル「Muse Spark」を発表 — Llama路線からの転換
Meta Superintelligence Labs（旧FAIR、Alexandr Wang率いる）が初のフロンティアモデル「Muse Spark」（コードネーム: Avocado）を発表。テキスト・音声・画像のマルチモーダル入力に対応し、ツール使用・ビジュアル推論・マルチエージェントオーケストレーションをネイティブサポート。「Contemplating」モードではエージェント群が並列推論し、Gemini Deep ThinkやGPT Proに対抗する。Llama系列のオープンウェイト路線から独自モデルへの転換が注目点。コーディング能力にはギャップを認めており、現時点では米国のみ提供。

- https://ai.meta.com/blog/introducing-muse-spark-msl
- https://www.cnbc.com/2026/04/08/meta-debuts-first-major-ai-model-since-14-billion-deal-to-bring-in-alexandr-wang.html

### Anthropic、控訴裁でペンタゴンのブラックリスト差し止めに敗訴
米連邦控訴裁判所（ワシントンDC）が、ペンタゴンによるAnthropicの「サプライチェーンリスク」指定の一時差し止めを却下。国防総省はClaudeの全用途無制限アクセスを要求、Anthropicは自律兵器・国内大量監視への利用制限を主張し対立。歴史上初めて米国企業がこの指定を受けた事例。訴訟は継続中。

- https://www.cnbc.com/2026/04/08/anthropic-pentagon-court-ruling-supply-chain-risk.html

### Anthropic「Claude Managed Agents」パブリックベータ開始
開発者がAIエージェントを構築・デプロイするためのクラウドサービス。サンドボックス実行環境、認証、チェックポイント、スコープ付き権限、永続セッションを提供。Notion・楽天・Asanaが初期採用。Claude Console、Claude Code、新CLIから利用可能。

- https://siliconangle.com/2026/04/08/anthropic-launches-claude-managed-agents-speed-ai-agent-development/

## AI開発ツール

### GitHub Copilot CLI「Rubber Duck」モード — クロスモデルレビューで精度74.7%向上
GitHub Copilot CLIに異なるモデルファミリーによるセカンドオピニオン機能を追加。メインモデルがClaudeならRubber DuckはGPT-5.4が担当。計画策定後・複雑な実装後・テスト作成後の3つのタイミングで自動的に批評。Claude Sonnet + Rubber Duckで、SonnetとOpus単体の性能差の74.7%を埋める結果。`/experimental`で利用可能。

- https://github.blog/ai-and-ml/github-copilot/github-copilot-cli-combines-model-families-for-a-second-opinion/

### Cursor 3リリース — エージェントファーストのインターフェース刷新
4月2日リリース。ローカル・ワークツリー・クラウド・リモートSSH環境で複数AIエージェントを並列実行可能。Agent Tabs、Design Mode（Cmd+Shift+D）、30以上のプラグイン（Atlassian、Datadog、GitLab、Hugging Face等）、マルチLLM比較、ネイティブマルチリポ対応。

- https://cursor.com/blog/cursor-3

## クラウド・インフラ

### Amazon S3 Files 一般提供開始 — S3バケットをファイルシステムとしてアクセス
S3バケットをNFS v4.1+経由でファイルシステムとしてマウント可能に。EC2、ECS、EKS、Lambdaから利用可。Amazon EFS技術ベースで、オブジェクトストレージの拡張性・耐久性とファイルシステムの操作性を両立。34リージョンで提供。AIエージェントやファイルベースアプリケーションのS3データアクセスが大幅に簡素化。

- https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-s3-files/

### AWS DevOps Agent 一般提供開始
AWS、マルチクラウド、オンプレミス環境のインシデント解決・予防、アプリケーション信頼性最適化、SREタスクを自動処理する常時稼働AIエージェント。

- https://aws.amazon.com/about-aws/whats-new/2026/03/aws-devops-agent-generally-available/
