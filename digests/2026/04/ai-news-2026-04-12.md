### ServiceNow: 全製品をAIネイティブ化、Context Engine と Build Agent Skills を発表
ServiceNowが4/9、「サイドカーAI時代の終焉」を宣言し、全製品ラインナップをAIネイティブに刷新。中核は新機能「Context Engine」で、Service Graph・Knowledge Graph・データインベントリ・ID関係・資産依存・データリネージを統合し、AIエージェントにリアルタイムの組織コンテキストを供給（プレビュー提供）。「Build Agent Skills」は4/15から、Claude Code / OpenAI Codex / Cursor / Windsurf / Antigravity 等のサードパーティツールでSkillsを開発し、ServiceNow AI Platformへ直接デプロイ可能に。AI・データ・セキュリティ・ガバナンスを別料金ではなく全製品の標準機能として内包する方針。Nvidia NeMo Agent Toolkit 採用（4/9ダイジェスト既出）に続くエンタープライズプラットフォーム陣営の動きとして、Microsoft・Salesforceとの正面衝突が鮮明に。

### Microsoft Agent Framework 1.0 一般提供開始 — AutoGen + Semantic Kernel 統合
Microsoftが4/3に Agent Framework 1.0 を .NET / Python 向けに正式リリース（GA）。「安定API・長期サポート」を明言。Semantic Kernel のエンタープライズ基盤と AutoGen のマルチエージェントオーケストレーションパターンを単一フレームワークに統合。A2A / MCP 経由のクロスランタイム互換、マルチプロバイダモデル対応（Microsoft Foundry / Azure OpenAI / OpenAI / Anthropic Claude / Amazon Bedrock / Google Gemini / Ollama を標準コネクタで提供）。2025年10月の初回発表、2026年2月RC を経てGA到達。社内エージェント開発で OSS AutoGen 系を採用していた企業にとって本番移行の最大の判断材料。
- https://devblogs.microsoft.com/agent-framework/microsoft-agent-framework-version-1-0/
- https://visualstudiomagazine.com/articles/2026/04/06/microsoft-ships-production-ready-agent-framework-1-0-for-net-and-python.aspx
- https://techcommunity.microsoft.com/blog/appsonazureblog/build-multi-agent-ai-apps-on-azure-app-service-with-microsoft-agent-framework-1-/4510017

### Microsoft Foundry Local が GA — PC / エッジでクラウド非依存のローカルLLM実行基盤
Microsoftが4/9、Foundry Local の一般提供開始を発表。クロスプラットフォーム（Windows / macOS / Linux）のローカルAI実行環境で、クラウド依存・ネットワークレイテンシ・トークン課金なしでデスクトップアシスタント・医療意思決定支援・プライベートコーディングアシスタント・オフラインエッジアプリを構築可能。Python / JavaScript / C# / Rust の SDK、Foundry Catalog からのデバイスハードウェア別最適化モデル自動ダウンロードに対応。Phi・Qwen（軽量チャット/推論）、Mistral・DeepSeek（コーディング/論理）、Whisper（音声文字起こし）を初期サポート。機密データをクラウドに送れない業界（医療・金融・公共）へのAI導入ハードルを大幅に下げる。
- https://devblogs.microsoft.com/foundry/foundry-local-ga/
- https://www.c-sharpcorner.com/news/foundry-local-is-now-ga

### Microsoft Agent Governance Toolkit を OSS 公開 — OWASP Top 10 for Agents 全対応
Microsoftが4/2に Agent Governance Toolkit を MIT ライセンスで公開。OWASP が2025年12月に公表した「Agentic AI Top 10」の全10リスク（ゴールハイジャック・メモリポイズニング・ローグエージェント等）に対応する初のランタイム実行時セキュリティツールキット。ポリシーエンジンはサブミリ秒（<0.1ms）で危険なエージェント動作を実行前にブロック。ゴールハイジャック対策としてセマンティック意図分類器を搭載。EU AI Act の高リスク義務（2026年8月施行）・Colorado AI Act（2026年6月施行）の対応準備と合わせて、エンタープライズがエージェントを本番運用する際のガバナンス選択肢が揃いつつある。
- https://opensource.microsoft.com/blog/2026/04/02/introducing-the-agent-governance-toolkit-open-source-runtime-security-for-ai-agents/
- https://www.helpnetsecurity.com/2026/04/03/microsoft-ai-agent-governance-toolkit/
- https://github.com/microsoft/agent-governance-toolkit/releases
