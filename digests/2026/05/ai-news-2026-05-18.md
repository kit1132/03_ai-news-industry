# AI News Digest: 2026-05-18

## AIプラットフォーム競争動向

### Anthropic、企業AI導入でOpenAIを初逆転 — Ramp AI Index 5月版
Ramp AI Indexの5月版で、Anthropic導入企業比率が34.4%に上昇（前月比+3.8pt）、OpenAIは32.3%に低下（-2.9pt）。AI競争開始以来、初めてAnthropicがビジネス導入率でOpenAIを上回った。Claude Codeが同社史上最速成長プロダクトとして寄与。
- https://venturebeat.com/technology/anthropic-finally-beat-openai-in-business-ai-adoption-but-3-big-threats-could-erase-its-lead

### Anthropic、$950B評価額で$30-50B調達交渉中
Anthropicが$950B（約142兆円）の評価額で$30-50Bの資金調達を交渉中。実現すればOpenAI（$852B）を超え世界最高評価の非公開企業に。年間ランレート収益は$30B超（2025年末の$9Bから3倍以上）、年間$1M以上支出する法人顧客は1,000社超。
- https://the-decoder.com/anthropics-900-billion-valuation-would-make-it-more-valuable-than-openai-for-the-first-time/

### Google I/O 2026（5/19開幕）直前プレビュー — 新Geminiモデル発表へ
5月19日のキーノートで新Geminiモデル発表見込み。性能はGPT-5.5相当だがAnthropic Claude Mythosには及ばないとの評価。「Gemini Intelligence」としてAndroidへのエージェント統合、動画生成「Gemini Omni」も予告。
- https://www.techtimes.com/articles/316755/20260517/google-i-o-2026-keynote-opens-tuesday-new-gemini-lands-behind-mythos-gpt-55.htm
- https://www.androidauthority.com/what-to-expect-from-google-io-2026-3664979/

## 開発者ツール・コーディングエージェント

### Microsoft、社内のClaude Codeライセンスを廃止 → GitHub Copilot CLIへ移行
MicrosoftのExperiences+Devices部門（Windows/M365/Teams等）が、6月30日を期限にClaude Codeの社内利用を終了。GitHub Copilot CLIへ一本化。Claude Codeが「人気になりすぎた」ことが背景。Anthropicの外部顧客向けサービスには影響なし。
- https://www.windowscentral.com/microsoft/microsoft-cancels-claude-code-licenses-shifting-developers-to-github-copilot-cli-a-move-likely-driven-by-financial-motives

### xAI「Grok Build」早期ベータ公開 — ターミナル型コーディングエージェント
Elon MuskのxAIがClaude Code / OpenAI Codex対抗のCLIコーディングエージェントを公開。サブエージェントによる並列処理、ローカルファースト（コード非送信）、plan-review機能を搭載。SuperGrok Heavy（$300/月、初期$99/月）で利用可能。
- https://x.ai/news/grok-build-cli
- https://the-decoder.com/x-ai-plays-catch-up-with-grok-build-its-first-terminal-based-coding-agent/

### OpenAI Codex、iOS/Androidモバイル対応（5/14）
ChatGPTモバイルアプリからCodexセッションの監視・承認・コンテキスト追加が可能に。Free/Goプラン含む全プランで利用可。週間アクティブユーザー400万人超。PCで起動したCodex環境をQRコードでモバイル連携。
- https://openai.com/index/work-with-codex-from-anywhere/

### VS Code「Agent Window」プレビュー公開
複数AIエージェントでの並行開発を容易にする新機能。エージェントごとに独立ウィンドウを割り当て、同時並行作業のコンテキスト分離を実現。
- https://www.publickey1.jp/blog/26/vs_codeaiagent_window.html

### Raindrop Workshop — AIエージェント向けOSSローカルデバッガ（5/14公開）
全トークン・ツールコール・判断をリアルタイムでローカルダッシュボードに表示。Vercel AI SDK/OpenAI Agents/Anthropic SDK/LangChain等主要フレームワーク対応。MIT License。
- https://venturebeat.com/technology/developers-can-now-debug-and-evaluate-ai-agents-locally-with-raindrops-open-source-tool-workshop
