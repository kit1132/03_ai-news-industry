### Google、Gemini CLI を提供終了——Antigravity CLI へ統合（6/18 実施）
6/18より、消費者向けのGemini CLIおよびGemini Code Assist IDE拡張がリクエスト提供を停止。Google AI Pro/Ultra・無料のGemini Code Assist個人版が対象。ターミナル型コーディングツールをエージェント特化開発基盤「Google Antigravity」（デスクトップ版2.0＋Go製Antigravity CLI）へ統合する。**エンタープライズ（Gemini Code Assist Standard/Enterprise、Google Cloud経由、有料APIキー）はGemini CLIを継続利用可。** Antigravity CLIはOSS版として公開されず、開発者から反発も。マルチエージェント・オーケストレーション前提の設計へ移行。
